Запуск скрипта:
```python table_parser.py```

Структура проекта:
```
   parser/
       ├── table_parser.py   # основной скрипт
       ├── config.json       # конфигурация
       ├── analytics.xlsx    # Excel-файл с данными
       │── cache.json        # создаётся автоматически для хранения дат
```

Логирование:
```
[SKIP] — если данные уже были отправлены ранее.

[INFO] — шаги процесса (отправка, отсутствие новых данных и т. д.).

Не найдено — если фильтр для строки не дал совпадений.
```

Восстановление окружения:
```
pip install -r requirements.txt
```
Ручная очистка кэша:
```
rm cache.json
```
